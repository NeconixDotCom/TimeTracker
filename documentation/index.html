Documentation

Time tracker

To start tracking, select your current work item from the issues list. The timer will start automatically and continue to run while the time tracker is open in your browser.

To stop tracking, select None as the current work item.


Issue logger

To log your time worked, find your work item in the Recent Work Items table and press Log. Follow the onscreen prompts. A comment will be added to the project issue showing the time worked.

You can also delete a work item without logging a comment on the project issue. Select Delete and follow the onscreen prompts.
